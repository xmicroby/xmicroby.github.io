## Terminal Intro Choreography

| Option | Description | Selected |
|--------|-------------|----------|
| Literal terminal with `init` commands | Shows a `>` prompt and fake commands before typing the name. | |
| Clean typewriter effect | Types the name/title directly with a blinking block cursor. | ✓ |

**User's choice:** "ooh that would be cool to have a type writer effect with a blinker. init commands are too much i think"
**Notes:** User chose the direct typewriter effect with a block cursor. Specifically requested the cursor be in the same color scheme as the rest of the UI.

## Waveform Behavior & Placement

| Option | Description | Selected |
|--------|-------------|----------|
| Ambient | Subtle, looping background that isn't too distracting. | |
| Mouse-reactive | Reacts to mouse/scroll to feel "alive". | |
| Tabled entirely | Do not build waveforms. | ✓ |

**User's choice:** "ambient. not too dsitracting... how is nlp like a wave? what kind of wave would LLMs be? note sure what this means... let's table the wave idea for now. the idea is interesting but not compelling enoguht"
**Notes:** The user initially chose "ambient", but correctly pointed out that LLM/NLP data isn't naturally a continuous wave. After discussing how to visually represent it, the user decided to table the waveform idea completely because forcing the metaphor wasn't compelling enough.
