**A. Add FOUC-prevention inline script to BaseLayout.astro (per D-04, D-05):**

Insert an inline `<script>` as the FIRST element inside `<head>`, immediately after the opening `<head>` tag and BEFORE the `<meta charset>` tag. This must be a blocking script (not `type="module"` or `defer`):

```html
<head>
    <script is:inline>
      (function() {
        var theme = localStorage.getItem('theme');
        if (!theme) {
          theme = window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light';
        }
        if (theme === 'dark') {
          document.documentElement.classList.add('dark');
        }
        // Add theme-ready class after a frame to enable transitions for subsequent toggles only
        requestAnimationFrame(function() {
          requestAnimationFrame(function() {
            document.documentElement.classList.add('theme-ready');
          });
        });
      })();
    </script>
    <meta charset="UTF-8" />
```

Key details:
- Uses `is:inline` (Astro directive to keep script inline, not bundled)
- Checks `localStorage.getItem('theme')` first (returning visitors)
- Falls back to `window.matchMedia('(prefers-color-scheme: dark)')` for system preference
- Falls back to light (no class added = `:root` light tokens apply)
- If dark, adds `.dark` class to `<html>` synchronously before first paint
- Uses double-`requestAnimationFrame` to add `.theme-ready` class AFTER first paint, enabling CSS transitions only for user-initiated toggles (not initial load)
- Also add `<meta name="color-scheme" content="light dark" />` after the viewport meta tag (best practice for browser UI hints)

Add an inline `<script>` at the bottom of Header.astro (after the closing `</header>` tag) for toggle logic:

```html
<script>
  function updateToggleIcons() {
    const isDark = document.documentElement.classList.contains('dark');
    document.querySelectorAll('.theme-icon-sun').forEach(el => {
      el.classList.toggle('hidden', !isDark);
    });
    document.querySelectorAll('.theme-icon-moon').forEach(el => {
      el.classList.toggle('hidden', isDark);
    });
  }

  // Set initial icon state
  updateToggleIcons();

  // Toggle handler for ALL theme toggle buttons (header + mobile nav)
  document.querySelectorAll('.theme-toggle').forEach(btn => {
    btn.addEventListener('click', () => {
      const isDark = document.documentElement.classList.contains('dark');
      if (isDark) {
        document.documentElement.classList.remove('dark');
        localStorage.setItem('theme', 'light');
      } else {
        document.documentElement.classList.add('dark');
        localStorage.setItem('theme', 'dark');
      }
      updateToggleIcons();
    });
  });
</script>
```
