Create `src/components/react/CardiacWaveform.tsx` — an SVG-based cardiac waveform visualization rendered from a synthetic data array, per D-05.

**Data generation:**
Create a static function `generateCardiacSignal(numBeats: number, samplesPerBeat: number): number[]` that produces a synthetic calcium transient waveform. The signal should mimic a real calcium transient trace:
- Rapid upstroke (depolarization phase) — steep rise over ~10% of beat duration
- Plateau/peak — brief hold at max amplitude
- Gradual decay (repolarization) — exponential-like descent over ~60% of beat duration
- Resting baseline between beats — flat at ~0.1 amplitude

Generate 3 complete beats with ~100 samples per beat (300 total data points). Store as a constant array at module level.

**SVG path generation:**
Convert the data array to an SVG `<path>` using a helper function `dataToSvgPath(data: number[], width: number, height: number): string`:
- Map data indices to x-coordinates linearly across the width
- Map data values (0-1 range) to y-coordinates (inverted — 0 at bottom, 1 at top) with padding
- Use `M` (moveTo) for first point, `L` (lineTo) for subsequent points
- This produces a polyline path — smooth enough at 300 points without bezier curves

**Component:**
```tsx
export default function CardiacWaveform() {
```

- Add a subtle grid background: horizontal dashed lines at 25%, 50%, 75% y-positions using `stroke="var(--color-border)"`, `strokeWidth="0.5"`, `strokeDasharray="4 4"`
